# Blinkit-Sales-Analytics
Interactive Power BI dashboard for Blinkit retail sales analysis.

## Overview
This project presents an interactive Power BI dashboard developed using the Blinkit retail sales dataset to analyze sales performance and business insights.

## Technologies Used
- Power BI
- Power Query
- DAX

## Key Features
- Interactive KPI cards
- Sales analysis by outlet type and size
- Product category analysis
- Customer rating analysis
- Interactive slicers and filters
